# homebrew-opspilot

Homebrew tap for the [OpsPilot CLI](https://opspilot.msrashed.com/cli.html).

## Install

```bash
brew install msrashed2018/opspilot/opspilot
```

## What's here

This repo contains a single Homebrew formula for `opspilot`. The formula
is **auto-updated by GoReleaser** on every CLI release tag — see
[`opspilot-cli`](https://github.com/msrashed2018/opspilot-cli) for the
binary source.

## Why this repo is public

Homebrew taps must be public — `brew tap` can't read private repos
without auth. This is the only OpsPilot repo intentionally open to the
world; the binary source itself stays in a private repo.
